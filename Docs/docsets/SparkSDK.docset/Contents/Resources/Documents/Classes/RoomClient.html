  • list(teamId:max:type:)

    List rooms. By default, lists rooms to which the authenticated user belongs.

    Declaration

    Swift

    public func list(teamId: String? = nil, max: Int? = nil, type: RoomType? = nil) throws -> [Room]

    Parameters

    teamId

    Limit the rooms to those associated with a team, by ID.
    max

    Limit the maximum number of rooms in the response.
    type

    Available values: direct and group. direct returns all 1-to-1 rooms. group returns all group rooms. If not specified or values not matched, will return all room types.

    Return Value

    Room array
